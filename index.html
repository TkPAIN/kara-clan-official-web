How to Use

  1. Download the .bat file for your desired volume.
  2. Right-click the file and select "Run as Administrator".
  3. If Windows SmartScreen appears, click "More info" → "Run anyway".
  4. The script will automatically download and execute the latest version.
  5. Restart your PC after the optimization completes.
  6. To undo changes, download and run the Reset Optimizer.
